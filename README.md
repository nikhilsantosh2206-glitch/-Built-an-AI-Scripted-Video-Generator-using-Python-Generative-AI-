# 🎬 Built an AI Scripted Video Generator using Python & Generative AI

## 🚀 Overview

This project is an AI-powered video generation system that automatically creates videos from a user-provided topic.

The application generates a script, converts it into speech, creates subtitles, combines images into a slideshow, and exports a final MP4 video automatically.

---

## ✨ Features

- 🧠 AI Script Generation
- 🎙️ Text-to-Speech Voice Generation
- 🖼️ Automatic Image Slideshow
- 🎞️ Video Creation with MoviePy
- 📝 Subtitle Generation
- 📦 MP4 Video Export
- ⚡ End-to-End Automation

---

## 🛠️ Tech Stack

- Python 3.11
- MoviePy
- gTTS
- FFmpeg
- OpenAI API (Optional)
- Pillow

## ⚙️ Installation

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```

Enter a topic:

```text
Future of AI in 2050
```

The system will:

1. Generate a script
2. Generate AI voice narration
3. Create subtitles
4. Build a slideshow video
5. Export final MP4 video
